🚀 PRIMEROS PASOS (5 MINUTOS)
═══════════════════════════════════════════════════════════════════════════

1. Lee esto:
   📄 COMIENZA_AQUI.md (orientación, 5 minutos)

2. Instala dependencias:
   $ cd chelonskysell-complete
   $ npm install

3. Inicia la app:
   $ npm run dev

4. Abre en navegador:
   http://localhost:3000

   ¡LISTO! La app estará funcionando
